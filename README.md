## 

- Project ini saya buat untuk membuat aplikasi yang memiliki forms seperti login ataupun register dengan adanya login dan register ini maka autentikasi terjalan dan seseorang jadi memiliki privasi

##
# Fitur utama
- formik dan yup = formik yang mengelola data dengan stau perintah tidak perlu membuat banyak state dan yup itu yang memvalidasi data seperti syarat password min atau max huruf dll dari formik.
- image packer = image packer untuk mengupload photo dari galery
- asynstorage = ini untuk membuat database local agak semua data yang di input dari form masuk ke dalam database dan tidak hilang walaupun apk ditutup

##
# Bonues level 3
- Password strength indicator (weak/medium/strong), 
- tab fokus otomatis saat tekan Next,
- simpan data ke AsyncStorage.
